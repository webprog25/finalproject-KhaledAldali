Webprog Final Project Proposal 
=============================

Project Title: Ernährungstagebuch  
Your Name: Khaled Aldali

Description
-----------
Ich will eine einfache Webseite bauen, auf der man seine Mahlzeiten eintragen kann.  
Man schreibt auf, was man gegessen hat, wie viele Kalorien es hat und wählt die Art (z. B. Frühstück, Mittagessen).  
Die Daten werden gespeichert und als Liste auf der Seite angezeigt. Man sieht auch die gesamte Kalorienanzahl.

Feature-Liste:
- Formular zum Eintragen von Mahlzeiten  
- Auswahl der Kategorie (Frühstück, Mittag, usw.)  
- Speichern in einer Datei am Server (JSON)  
- Alle Einträge werden angezeigt  
- Seite lädt neue Einträge automatisch nach

Milestone(s)
------------
HTML und CSS sind fertig. Die Seite sieht gut aus und das Formular funktioniert.  
JavaScript ist noch nicht ganz fertig, aber es speichert schon neue Einträge.  
Der Server mit Express funktioniert. Ich kann Daten senden (POST) und abfragen (GET).
